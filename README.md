# docxload
> 通过标签配置实现word文档前端导出

## 引入
```node
npm install --save docxload
```

## 使用
```js
import exportWord from 'docxload

let template = `
  <page>
    <p>hello world</p>
  </page>
`

exportWord(template)
```

## 标签配置
